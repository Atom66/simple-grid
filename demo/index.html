simple-grid: The Demo.

simple-grid is an editable table for AngularJS in the form of a directive.

Grid options used in this demo: 

{{pretty(gridConfig.options)}}

Prepopulated: <div simple-grid="gridConfig"></div>

The data model for the above grid: {{data}}

Empty: <div simple-grid="gridConfigEmpty"></div>

Config is same as above, `getData` returns an (iniitally) empty list.
